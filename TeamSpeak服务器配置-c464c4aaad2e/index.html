TeamSpeak 服务器配置

发表于2025-07-31|更新于2025-09-06|Server
|总字数:442|阅读时长:1分钟

1.Windows 安装 TeamSpeak 服务器

下载服务器文件

  • 访问官方地址：https://www.teamspeak.com/en/downloads/#server
  • 选择 Windows 版本并下载。

解压并运行

  • 解压下载的 .zip 文件到一个文件夹（例如 C:\TeamspeakServer）。
  • 运行 ts3server.exe。
  • 第一次运行会生成：
    • Server Admin Token（第一次连接时用于获取管理员权限）
    • Server Query Admin Account（命令行管理账号）

允许防火墙访问

  • 弹出 Windows 防火墙提示时，点击 允许访问。
  • 如果防火墙没弹出，可以手动开放端口（默认 UDP 9987、TCP 10011、TCP 30033）。

连接到服务器

  • 下载 TeamSpeak 客户端：https://www.teamspeak.com/en/downloads/#client
  • 连接地址输入你的电脑 IP（或公网 IP）。
  • 输入刚才的 Admin Token 获取管理员权限。

2.Linux平台（以 Ubuntu/Debian）

首次启动时，终端会显示 Server Admin Token，请保存好。

端口默认：

  • UDP 9987（语音）
  • TCP 10011（ServerQuery）
  • TCP 30033（文件传输）
